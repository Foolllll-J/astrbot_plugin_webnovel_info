# 起点小说信息查询插件

AstrBot 起点小说信息查询插件，支持搜索起点小说并获取详细信息。

## 功能特性

- **搜索功能**：支持按书名搜索起点小说
- **分页浏览**：搜索结果支持分页浏览，每页显示20个结果
- **书籍详情**：查看小说的详细信息（书名、作者、简介、评分、字数等）
- **显示模式**：支持简洁模式和详细模式两种显示方式
- **命令别名**：`/qd` 是 `/起点` 的别名
- **智能匹配**：当搜索结果唯一且书名完全匹配时，直接返回书籍详情

## 使用方法

### 搜索小说

```
/起点 <书名>
/qd <书名>
```

### 翻页浏览

```
/起点 下一页
/qd 下一页
/起点 上一页
/qd 上一页
```

### 查看书籍详情

```
/起点 <序号>
/qd <序号>
```

## 配置选项

- `display_mode`: 显示模式，可选值：
  - `detailed`：详细模式，显示所有可用信息
  - `concise`：简洁模式，只显示封面、书名、作者、字数、简介

## 示例

搜索小说：
```
/起点 凡人修仙传
```

翻页：
```
/qd 下一页
```

查看详情：
```
/起点 1
```
